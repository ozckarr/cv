I detta skolarbete skulle ickeplatser uppmärksammas med en mobilapp. Ickeplatser förklaras kvickas som stadens mellanrum. Det ska tillläggas att detta skedde en månad innan någon visste något om Pokémon Go. Vi valde upplägget då AR-spel utbudet var obruten mark.
Arbetet utfördes tillsammans med Emilia Skoog, Hanna Eliasson och Géza Ribberström Palyi.

Textbaserat äventyr om potatis. Involverar en hel del trams. Svaren måste ske i text, då övningen var att hantera felaktig input. Jag valde att göra det på många olika vis.

GitHub

Uppgiften var att rätta till ljudboksspelares snabbspolingknapp via fjärrkontroll som hoppar i kapitel vilket gör det svårt att hitta tillbaka om man kommer åt den. Här är tre förslag.

1: Ändra kapitelhopp till 15-sekunders hopp så att man fortfarande kan manövrera.
2: Snabbspolningar man så sätts ett bokmärke där hen var. Klickar man tillbaks så kommer man dit man var senast. Allt sparas sa att man även kan bokmärka utan att stanna upp.
3: Kapitelhopp måste bekräftas med ännu ett klick. Ljudboken låter utifall att man omedvetet kommit åt.
